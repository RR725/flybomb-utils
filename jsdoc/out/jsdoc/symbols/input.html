方法详细说明

  • <static> input.placeholder(options)
    聚焦去掉提示文字，失焦显示提示文字 
    input.placeholder({
                  id:'#test',
                  text:'密码',
                  color:['#000','#999'],
                  focusCallback:function(){},
                  blurCallback:function(){}
              })
    Parameters:
    {Object} options
    参数集合
    {String} options.id
    必选，输入框的id名
    {String} options.text
    必选，输入框默认显示的文字
    {Array} options.color
    必选，输入框里的文字颜色，输入文字和默认提示文字
    {Function} options.focusCallback
    可选，focus的回调函数
    {Function} options.blurCallback
    可选，blur的回调函数
  • <static> input.setCursorPos(options)
    与input.focus()的区别是，这里捕获异常，并且获得焦点后光标默认在文本最后
    Parameters:
    {Object} options
    参数集合
    {Object} options.dom
    获得焦点的输入框
    {Number} options.pos
    光标的位置 0:最前,1:选中所有文本,2:最后,默认为2
  • <static> input.textToEdit(options)
    点击文字出现输入框，可修改文字 
    input.textToEdit({
                  element:'data-toedit',
                  cssClass:'xxxxxx',
                  type:'textarea',
                  blurCallback: function(e) {
                    var target = $(e.target);
                    target.hide();
                    target.prev().show().text(target.val());

                    something();
                    .......
                  }
              })
    Parameters:
    {Object} options
    参数集合
    {String} options.element
    必选，要进行编辑的标签属性 data-toedit
    {String} options.cssClass
    必选，控制输入框的样式
    {String} options.type
    可选，输入框的类型 input or textarea 默认input
    {Function} options.blurCallback
    可选，blur的回调函数
